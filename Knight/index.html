The Basics

A PASSWORD IS ONLY AS SECURE AS ITS ENCRYPTION

Creating a Password

Passwords are usally created following specific regulations from each website. However, these requirements are often not enough.

Hashing a Password

In terms of passwords, a hash is an encrypted version of the origional password. When a hashing algorithm is applied to a password, an irreversable mathematical algorithm is used to convert your password to a uniform lenth of characters. It is impossible to reverse a hash, but it is possible to use a brue-force algorithm to try all of the possiblilites. For this example we used the md5 hashing algorithm.

Brute Force Algorithm

Brute-force, exhaustive search, or generate and test, is a very general problem-solving technique that consists of systematically generating a password and it's respective hash until the correct hash is found.

Breaking a Hash

To properly "break" a hash, a possible password is generated and then the hasing algorithm is applied. That test password's hash is then compared to the hash you want to "break" until the correct hash is found.
